# Proposed Color Mapping

Cool Colors for Plan Agents (calming, thoughtful):
- Blue family: #4A90E2 (sky blue), #5B9BD5 (soft blue), #6CA6CD (steel blue)
- Green family: #50C878 (emerald), #66CDAA (aquamarine), #5F9EA0 (cadet blue)
- Purple family: #9370DB (medium purple), #8A7FC4 (lavender), #7B68EE (medium slate blue)

Warm Colors for Build Agents (energetic, action):
- Red family: #E74C3C (crimson), #DC143C (carmine), #FF6B6B (coral red)
- Orange family: #FF8C42 (mango), #FFA500 (bright orange), #FF7F50 (coral)
- Yellow family: #FFB347 (pastel orange), #FDB813 (golden), #FFAA33 (orange yellow)
